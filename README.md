# Algorithms and Data Structures

A collection of lab projects completed during coursework for CSCI 112, a second course in programming and problem solving with Python at the undergraduate level. This course builds on foundational programming knowledge with a focus on algorithm analysis, abstract data types, and the design and implementation of core data structures.

## Course Topics

- **Algorithm Analysis** – Big-O notation and techniques for evaluating algorithmic efficiency
- **Abstract Data Types** – Formal properties, interfaces, implementations, and real-world applications of collections and ADTs
- **Data Structures** – Arrays, Linked Structures, Trees (Binary, Red-Black, B-Trees), Heaps (Binary, Leftist, Binomial), and Graphs (Adjacency Lists, Adjacency Matrices, Edge Lists)
- **Algorithms** – Searches, Sorting, Graph Searches, and Flow/Cut algorithms

## Repository Structure

Each directory corresponds to a lab assignment and contains the Python source code along with any supporting files or write-ups.

```
csci112-labs/
├── lab01/
├── lab02/
├── ...
└── labN/
```

## Requirements

- Python 3.10

## Usage

Clone the repository and navigate to any lab directory to run the code:

```bash
git clone https://github.com/wonjunjo123/csci112-labs.git
cd csci112-labs/lab01
python solution.py
```
